Exceptions:

EVarLoaderError

Base class for this projects errors.

InvalidEnvironmentVariablesError

Thrown when environment variables are found missing.

MissingEnvironmentVariablesError

Thrown when environment variables are found missing.

Classes:

EnvironmentVariable

Defines an Environment Variable to handle.

EnvironmentVariableLoader

This is the container for your EnvironmentVariable definitions, along with their eventual loaded config values.

Functions:

comma_separated_str_to_list(config_val)

Splits a comma-separated environment variable into a list of strings.

comma_separated_to_set(config_val)

Splits a comma-separated environment variable into a set of strings.

value_to_bool(config_val)

Massages the 'true' and 'false' strings to bool equivalents.

value_to_int(config_val)

Convert the value to int.

value_to_none(config_val)

Given a value that evaluates to a boolean False, return None.

validate_is_boolean_true(config_val, evar)

Make sure the value evaluates to boolean True.

validate_is_not_none(config_val, evar)

If the value is None, fail validation.
